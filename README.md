# 探検レーダー (Kobe Radar)

This is the official website for 探検レーダー, a location-based exploration app for Kobe, Japan.

## About

Explore 10 famous landmarks in Kobe using only a radar UI showing direction and distance. Collect stamps and take commemorative photos as you discover the city.
